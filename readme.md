
* 技术架构:`node.js` + `express` + `mongodb` + `session` + `socket.io`<br />
* 本项目主要使用了`html` + `css` + `js` + `jquery`进行页面布局和设计,用`node.js` 及 `express`<br />框架搭建后台服务器。使用`mongodb`数据库存储数据<br />
